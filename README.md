# Table-Handling-with-Selenium
This project demonstrates how to automate web table interactions using Selenium WebDriver in Java. The test focuses on extracting table metadata and row values

Launches the practice page in Chrome.
Locates the table with id = "product".
Prints the total number of rows in the table.
Prints the number of columns in the header row.
Extracts and prints all cell values from the second row of the table.
